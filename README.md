1) What is the difference between null and undefined?
Answer: null হচ্ছে variable এ default ভাবে value assign করা হই। undefined হল যখন কোনো variable declare করা হই কিন্তু value assign kora হই না।

3) What is the difference between == and ===?
Ans: == হল value চেক করে এবং === value and data type চেক করে।

5) Explain the concept of Scope in JavaScript (Global, Function, Block).
Ans: JavaScript এর scope হলো কনো variable কোথা থেকে access করা যাবে। 
Global Scope: Global a Scope এ declare করা variable , program এর সব জাইগা থেকে access করা যায়।
Function Scope: শুধু Function er ভেতর declare and use করা যায়।
Block Scope: {} bracket এর ভিতর declare করা variable শুধু ওই block এর ভিতর কাজ করে।
